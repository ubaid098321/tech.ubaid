Lab Setup for Ethical Hacking

Step 1: Install Termux

Termux ko Play Store se install mat karo. Latest aur secure version ke liye F-Droid se download karo.

Step 2: Basic Termux Commands

  • pkg update && pkg upgrade - System update karo
  • pkg install git - Git install karo
  • pkg install python - Python install karo
  • pkg install nano - Nano text editor install karo
  • termux-setup-storage - Storage access enable karo

Step 3: Clone Your GitHub Repo

git clone https://github.com/ubaid098321/tech.ubaid.git
cd tech.ubaid

Step 4: Run Your Website Locally

Simple Python HTTP server se apni website ko local host pe run karo:

python -m http.server 8080

Phir browser me jaake http://localhost:8080 open karo.

Step 5: Install Useful Tools

  • pkg install nmap - Network scanner
  • pkg install openssh - SSH tools
  • pkg install curl - Command line HTTP requests
